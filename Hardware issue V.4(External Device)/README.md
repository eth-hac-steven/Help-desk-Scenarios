# Hardware issue V.4 (External Device)

This contains all user-issue of involving external Device like 
- Keyboards
- Mouse
- Monitor

This user ticket may include 
- Keyboard not working -ticket
- Mouse not working -ticket
- Monitor not working -ticket

This kind of ticket have little detail when submitted by these Users so the "Not working" is very different,

### Keyboard/Mouse not working 
| Possible Issue | Solutions |
| ---------------|-----------|
| The keyboard/Mouse was never connected to the CPU | connect it|
| The keyboard/Mouse is wireless(bluetooth)| connect it |
| The keyboard/Mouse does not work on a system but works on others | - check the driver in Device manager, Disable and re-enable then test, -restart the system |
| The keyboard/Mouse is really damaged | Replace with a new one from inventory |

### Monitor not working
| Possible Issue | Solutions | 
| ---------------|-----------|
| The Power/display/HDMI cable are either damaged or not properly pluged in | Replace or re-seat them|
| The Monitor is really damaged | Replace with a new one from inventory |
